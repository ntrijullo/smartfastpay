# SmartFastPay Laravel Project

## Descripción

Este es un proyecto de Laravel. A continuación, se detallan los pasos básicos para instalar, configurar y ejecutar el proyecto.

## Requisitos

- PHP 8.0 o superior
- Composer
- MySQL o cualquier otra base de datos compatible
